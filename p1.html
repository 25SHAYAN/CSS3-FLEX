CSS FLEX

THE FLEXBOX OR FLEXIBLE BOX MODEL IN CSS IS A ONE-DIMENSTIONAL LAYOUT MODEL THAT HAS FLEXIBLE AND EFFICIENT LAYOUTS WITH DISTRIBUTED SPACES.

FLEXIBLE BOX LAYOUT MODULE,MAKES IT EASIER TO DESIGN FLEXIBLE RESPONSIVE LAYOUT STRUCTURE WITHOUT USING FLOAT OR POSITIONING

PROPERTIES FOR THE PARENT(FLEX CONTAINER)

1)display:flex;

2)flex-direction:row;

3)flex-wrap:wrap

4)flex-flow:row wrap; (flex-flow=direction+flex-wrap;)

5)justify-content:center;

6)align-items:center;

7)align-content:space-between;

8)column-gap:20px;

9)row-gap:20px;

10)gap:20px;

PROPERTIES FOR THE CHILDREN(FLEX ITEM)

1)flex: 1; or flex-grew: 1;

2)flex-shrink:3;

3)order:2;

4)align-self:center;
